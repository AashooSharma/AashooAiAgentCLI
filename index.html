Every high-risk action (modifying files, running shell commands) needs your go-ahead. Choose Allow, Always Allow, Deny, or Deny with a reason that teaches the agent an alternative approach.

WORKFLOW

How AashooAgent Works

A transparent, permission-based cycle that keeps you in control at every step.

01

You Describe the Task

Type your request in natural language. Use multiple lines freely — press Enter to add lines, Ctrl+S to send. Paste long prompts without issues.

→
02

Agent Creates a Plan

For complex tasks, the agent generates a detailed markdown plan — files to create, steps to execute. You can approve, edit, or completely redirect it.

→
03

Permission-Gated Execution

The agent executes step by step. For risky actions it asks permission. Low-risk reads run automatically. You are never surprised by unauthorized changes.

→
04

Remembers & Continues

All conversations, your tool preferences, and agent notes are saved per-project. Open the project tomorrow and the agent picks up with full context.

⚠️ Permission Required
✏️ edit_file — { "path": "app.py", "old_code": "...", "new_code": "..." }
[A] Allow [!] Always Allow [D] Deny [R] Deny + Reason [X] Stop Agent

Choose [R] Deny + Reason and the agent will read your reason and try a different approach automatically.

Installation Guide

Choose your platform and follow the steps. You'll be running your first AI coding session in under 5 minutes.

Prerequisites: Install Termux from F-Droid (recommended over Play Store for latest version).
1

Install Dependencies

pkg update -y && pkg install python git ripgrep termux-api -y
2

Clone the Repository

git clone https://github.com/AashooSharma/AashooAiAgentCLI.git
cd AashooAiAgentCLI
3

Setup Virtual Environment & Install

python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
4

Run Setup Wizard

python -m aashoo.main --setup
5

Start the Agent

python -m aashoo.main

DOCUMENTATION

Slash Commands Reference

Once inside a session, these commands give you full control without leaving the terminal.

/help

Show all available commands.

/switch

Dynamically switch LLM provider, model, or API key — mid-session.

/editor

Launch a Monaco-based Web Editor in your browser for visual editing.

/history

View the last 20 messages from the current project's conversation.

/undo

Revert the last file change made by the agent instantly.

/tree

Refresh and reprint the git-status colored live file tree.

/bg

List all currently running background processes (servers, jobs).

/bg-stop N

Stop background process by its ID. E.g. /bg-stop 1.

/clear

Clear the terminal screen while keeping your session active.

/exit

Exit the current project session and return to the main menu.

⌨️

Multi-line Input Support

Press Enter to add a new line while typing your prompt. When you're done, press Ctrl + S to send. Paste large multi-line prompts directly — they won't submit until you press Ctrl+S.
