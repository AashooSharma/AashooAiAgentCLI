CORE ADVANTAGES

Packed with Agentic Capabilities

Designed to be extremely lightweight, lightning fast, and highly secure. Everything a modern developer needs to automate code modifications.

USP Highlight
📱

Termux & Android Native

Write, test, and run code directly on your Android phone via Termux. Lightweight, optimized for arm64/mobile devices, and doesn't require any server-side computing resources.

💸

100% Free & Open Source

Licensed under the permissive MIT License. No subscription fees, no premium tiers, no usage caps, and absolutely no telemetry or cloud logging. You own your code.

🧠

Plan-First Reasoning

For complex modifications, the agent creates a step-by-step implementation plan. Review, customize, or suggest alternative designs before the agent executes code.

🔒

Permission Control System

Total transparency. The agent explicitly requests permission for modifying files, calling external APIs, or executing shell processes. You can Allow, Deny, or specify Deny+Reason.

🔄

Git-Status colored File Tree

Features a real-time, git-aware visual file tree that updates instantly as files are modified. Green for additions, yellow for edits, and red for deletions.

⛓️

Multi-LLM & Key Rotating

Supports Google Gemini, Groq, Anthropic Claude, OpenAI, and local Ollama. Input multiple API keys per provider to automatically bypass rate limits dynamically.

GETTING STARTED

Install in 3 Simple Steps

Get up and running in less than 2 minutes. Select your target environment below.

01

Prepare Environment

Ensure you have Python 3.9+ and git installed. Run the command block to update packages and set up dependencies.

# Termux dependencies
pkg update -y && pkg install python git ripgrep termux-api -y
python -m venv venv && source venv/bin/activate
02

Install Package & Configure

Install AashooAgent from PyPI and run the interactive configuration wizard to configure models and set up API keys.

pip install aashoo-agent
python -m aashoo.main --setup
03

Start Coding!

Open your project directory and boot up the interactive multi-line terminal session with your agent.

python -m aashoo.main
